# OPL Health Tools

OPL Health Tools 是 OPL Health Platform 的医疗工具能力层。

它把院内系统、数据库、文献源、分析工具、报告工具和科室接口组织成可授权、可审计、可用于智能体任务的工具包。

## 核心内容

- HIS、EMR、LIS、PACS 等院内系统接入。
- 指南库、文献库和科研数据库。
- 结构化数据库、文件系统和对象存储。
- 统计分析、图表、文档、报告和幻灯片工具。
- 科室或医院内部接口。

## 工具包记录

每个医疗工具包应记录：

- 工具名称。
- 数据范围。
- 权限要求。
- 可执行动作。
- 输入输出格式。
- 调用记录要求。
- 负责人和审批记录。

## 工具分层

| 层级 | 示例 | 目标 |
| --- | --- | --- |
| 数据工具 | 数据库、文件、对象存储 | 提供受控数据访问 |
| 医学工具 | 指南库、文献库、术语库 | 提供专业资料来源 |
| 分析工具 | 统计、图表、报告生成 | 支持科研和质控交付 |
| 系统工具 | HIS、EMR、LIS、PACS | 对接院内业务系统 |

## 与 OPL Cloud 的关系

OPL Health Tools 通过 OPL Fabric 的连接器和执行适配器进入工作空间。OPL Console 管理授权策略，OPL Ledger 记录关键调用和产物来源。
