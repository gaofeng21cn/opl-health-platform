# OPL Health Review

OPL Health Review 是 OPL Health Platform 的医学审查与证据能力层。

它为关键任务提供来源、规则、工具、输出、审查和责任记录，帮助医院把医疗智能体结果纳入可复查、可管理、可交接的工作流程。

## 审查对象

| 对象 | 审查重点 |
| --- | --- |
| 医学知识包 | 来源、版本、适用范围、更新机制 |
| 临床规则包 | 规则依据、使用场景、人工确认点 |
| 医疗工具包 | 权限、数据范围、调用记录 |
| 专病模板 | 输入要求、输出格式、审查路径 |
| 医疗智能体 | 能力边界、失败模式、版本变化 |
| 任务交付物 | 来源、计算、引用、审查结果 |

## 审查记录

重要任务应保留：

- 输入来源。
- 使用的知识包和规则包。
- 调用的工具和运行环境。
- 输出产物。
- 审查结果。
- 负责人。
- 后续继续入口。

## 审查类型

| 类型 | 说明 |
| --- | --- |
| 医学来源审查 | 检查指南、文献、制度和资料来源 |
| 临床规则审查 | 检查规则适用范围和人工确认点 |
| 数据与工具审查 | 检查权限、数据范围和调用记录 |
| 交付质量审查 | 检查引用、计算、图表、格式和结论边界 |

## 与 OPL Cloud 的关系

OPL Health Review 使用 OPL Ledger 记录任务回执、来源、审查结果和继续入口。医学专业判断由医院指定的专家、科室或平台负责人确认。
